List maker

Features builder

  • Instructions

    1. Choose a template
    2. Make a spreadsheet.
      • Example list spreadsheet
    3. Make it public
      (File->Publish to Web)
    4. Copy the document link
    5. Paste the document link into url field on right
    6. Click create
    7. Start a new feature in Chorus
    8. Click html view
    9. Copy and paste html from right into Chorus

  • Tips

    You can edit in Chorus or you can redo this everytime you want to make a change. Best to choose one.

    If you toggle back and forth between html and visual view you're going to have a bad time.
